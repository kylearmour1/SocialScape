# SocialScape

    
  [![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
  
  

  ## Description
A RESTful API for a social network web application where users can share their thoughts, react to friends' thoughts, and create a friend list. The API is built with Node.js, Express.js, MongoDB, and Mongoose.

  ## Instalation

    1. Clone this repository to your local machine.
    2. Ensure MongoDB is installed on your machine. Follow the MongoDB installation guide to install MongoDB locally.
    3. Install the required dependencies by running npm install in the project's root folder.


  ## Usage

  1. Start the server by running npm start in the project's root folder.
  2. Use Insomnia to test the available routes.
